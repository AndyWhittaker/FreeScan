

                        DATA STREAM A135  SPECIFICATION

       SPECIFICATIONS FOR DATA STREAM INFORMATION
       ------------------------------------------

       ENGINE   USAGE:
              2.0L OHC TBI   LT2  VIN = K       1990  J CAR
              2.0L OHC TBI   LT2  VIN = K       1991  J CAR

       DATA PIN  : READ DATA ON PIN "M" OF ALDL CONNECTOR

       BAUD RATE : 8192 BAUD - UNI-DIRECTIONAL

..PAGE
       NORMAL MODE MESSAGES (MODE 0)


                     *F96* MESSAGE
                    ----------------
                ID = $14
                MESSAGE LENGTH = 85 + 22 = 107 = $6B
                DATA 1
                   .

                   .
                DATA 22
                CHECK SUM


                     *F95* MESSAGE
                    ----------------
                ID = $F0
                MESSAGE LENGTH = 85 + 64 = 149 = $95
                MODE 1
                DATA 1
                   .
                   .
                   .
                DATA 63
                CHECK SUM
